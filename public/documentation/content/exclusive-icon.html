Exclusive Icon

Guidance and suggestions for using our exclusive set of icon libraries.

Overview

MyTravel icons are designed exclusively designed on top of premium icon set to make MyTravel pages unique, which means that these icons are only available in the MyTravel Template. This set of icons include more than 100 customized icons and counting.

Icons can be found in assets/fonts

How to use?

Copy-paste the following <css>s near the end of your pages under CSS Implementing Plugins to enable them.

                
                 <link rel="stylesheet" href="../../assets/css/font-mytravel.css">
